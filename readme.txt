== Description ==

This simple plugin has a widget that displays the most recent posts for a Wordpress site.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `simple-recent-posts` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Drag the 'Simple Recent Posts' widget to a widget area and configure to your pleasing.
